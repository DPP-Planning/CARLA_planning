The easiest way to run this project is using Docker. No need to install Python, CARLA, or any dependencies!

### Quick Start

**Windows:**
```batch
run-docker.bat up
```

**Linux/Mac:**
```bash
chmod +x run-docker.sh
./run-docker.sh up
```

That's it! The CARLA server and your planning code will start automatically.

### Prerequisites
- Docker Desktop (Windows/Mac) or Docker Engine (Linux)
- 8GB+ RAM
- 10GB free disk space
